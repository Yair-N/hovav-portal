## Architecture

- **Frontend:** Next.js 15 + Tailwind, RTL Hebrew, deployed from `hovav-portal`
- **CMS:** Sanity Studio (project ID `wprns757`, dataset `production`)
- **Data flow:** All pages fetch from Sanity via GROQ queries in `src/sanity/queries.ts`. The Sanity client is configured in `src/sanity/client.ts` using env vars from `.env.local`
- **Revalidation:** Homepage uses `revalidate = 60` (ISR). Other pages fetch on each request

## What's done

### Sanity integration
- All 10 section pages fetch from Sanity (news, games, jokes, comics, thoughts, opinions, recipes, recommendations, behind-the-scenes, sports highlights)
- Homepage fetches featured news, section previews (flexible hour + culture), and sports highlights from Sanity
- Sanity Studio has schemas for all content types + a `siteSettings` singleton for homepage section previews
- Image pipeline works: Sanity image refs → `cdn.sanity.io` URLs via `@sanity/image-url`
- `next.config.ts` allows `cdn.sanity.io` as a remote image host

### Frontend
- Homepage: 6 sections matching Canva design (hero, news, flexible hour, behind-the-scenes, culture, sports)
- Hero: corkboard image (background-removed PNG), title, tagline with microphone icon, thin black bar, CTA button
- All section pages have `PageHeader` component (blue band + image + title + thin black `h-4` bar)
- Mobile nav component (`MobileNav.tsx`)
- Fonts: Gan Bold (headings), Ktav Yad (handwriting tagline), Rubik (body)
- Color palette: `--hovav-header-blue`, `--hovav-sports-gray`, `--hovav-footer-beige`, `--hovav-cream`
